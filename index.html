Exploration of block-level elements and their default postion.

Here we will talk about how block-level elements can create a block that fills
the entire width of their parent elements, and they prevent other elements from
appearing in the same horizontal space.
The default position can be changed by setting its position property:


  • Static-Default Value
  • Relative Value
  • Absolute Value
  • Fixed Value
  • Sticky Value

Static Default Value

With the static default value, the block level element creates a block the full width of the parent element, and prevents other elements from appearing in the same horizontal space. block -level elements also appear on the left side if the browser. CSS syntax code used for default: 

        .default_rectangle {
             color: bisque;
            background-color: coral;
        }

This is Default


Relative Value

Using the relative value, will allow you to position an element relative to it's default static position. This can not just be done by simply putting the position status equal to relative. In order to do this it needs to be accompanied by an offset propety so we know how to we will make the position relative. There are 4 properties we can choose from to offset the element with:

  • Offset variables
  • Top: This moves the element down from the top
  • Bottom: This moves the element up from the bottom
  • Left: moves elements away from the left.
  • Right: moves elements away from the right.

Looking at the offset properties, the name describes as to where you want to move it away from. If only we could make it as simple when it comes to life and we could make our offset value equal to parents. So, how to practice writing the syntax within css for this relative value: ex: 


        .relative_rectangle {
            background-color: blanchedalmond;
            color: green;
            position: relative;
            top: 50px;
            left: 120px;
        }
below you will see how this code looks in comparison to the default.

This is Default

This is Relative and it is moved from the top by 50px, and moved from the left with 120px.


This is Absolute Value

With the absolute value it will ignore elements and act as if it isn't present on the page. The element within the absolute position will be relative to its closest parent element. When a person scrolls, the element will scroll with the rest of the document when a user scrolls. let me also introduce you to the concept of z-index
for this code I used the z-index controls how far back or forward an element is on the page. with a z-index of 1 it moves the element forward;
whereas 0 pushes it back.


The code I used for absolute value: 

 .absolute_rectangle {
                background-color: greenyellow;
                color: blue;
                position: absolute;
                top: 62%;
                left: 29%;
                transform: translate(-50%, -50%);
                z-index: 1; /* Adjust this value if needed */
            
            }



This is Absolute



Fixed Value

The fixed element can set an element to a specific positon on the page. this is good for setting the positon and that it does not scroll with the document such as with the absolute value. This also is accompanied with the same offset values as with relative

CSS syntax I used for this example:

        .fixed_rectangle {
    background-color: gold;
    color: black;
    position:fixed;
    bottom: 70px;
    top: 580px;
}

This is Fixed



Sticky Value

With sticky position value that keeps an element in the document flow as the user scrolls, but sticks to a specific position. When we use the sticky value we will also be using the familiar offset values.

CSS Syntax used for this example

        .sticky_rectangle {
            background-color: darkorange;
            color: white;
            position: sticky;
            bottom: 20px;
        }

This is Sticky
